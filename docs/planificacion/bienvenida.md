## 📋 Antes de Empezar (Pre-requisitos)
Para que las clases sean 100% prácticas, asegúrate de tener:
1.  **Cuenta de Google (Gmail)** activa.
2.  **Google Chrome** instalado.
3.  Acceso a la versión paga de **ChatGPT (Plus)** o la versión gratuita de **Gemini**, pero te recomiendo tener al menos una herramienta de pago para ver el potencial real.
4.  Un documento o reporte complejo (PDF) que uses en tu trabajo diario para practicar el análisis.
